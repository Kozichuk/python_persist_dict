Persisted Dictionary
====

[![PyPi Version][pypi-image]][pypi-url]


Dictionary implementation with saving values into files. 
Based on shelve lib.

Supported operations
--
- get value by key
- set value by key
    - if key exists - rewrite value
- delete value by key
- clear dictionary
